# async

# 语法

async函数返回一个 Promise 对象，可以使用then方法添加回调函数。当函数执行的时候，一旦遇到await就会先返回，等到异步操作完成，再接着执行函数体内后面的语句。

# 返回Promise

async函数内部return语句返回的值，会成为then方法回调函数的参数。

# await

正常情况下，await命令后面是一个 Promise 对象。如果不是，会被转成一个立即resolve的 Promise 对象。

# 注意点

  • await命令后面的Promise对象，运行结果可能是rejected，所以最好把await命令放在try...catch代码块中。
  • 多个await命令后面的异步操作，如果不存在继发关系，最好让它们同时触发。
let foo = await getFoo();
let bar = await getBar();

# example

定义一个异步函数,传入一个时间，单位毫秒，返回一个promise对象，在time秒状态后置为成功。

let timeout = function (time) {
  return new Promise((res,rej)=>{
    setTimeout(() => {
      res(time)
    }, time);
  })
}

# 串行，需要用时5s

!(async () => {
  console.time('test1');

  const t1 = await timeout(1000)
  const t2 = await timeout(4000)

  console.log(t1);
  console.log(t2);
  
  console.timeEnd('test1');
  
})()

# 循环中的并行

!(async () => {
  console.time('test4');

  const times = [1000,4000]
  const timesPromises = times.map(v=>timeout(v))
  for (const promise of timesPromises) {
    res = await promise
    console.log(res);
  }
  
  console.timeEnd('test4');

# 并行，按次序输出

!(async () => {
  console.time('test3');

  const times = [1500,1000,2000]
  const res = times.map(async(v)=>{
    return await timeout(v)
  })
  for (const v of res) {
    console.log(await v);
  }
  
  console.timeEnd('test3');
  
})() 
//会等1.5s输出1500，1000，再等500ms，输出2000
